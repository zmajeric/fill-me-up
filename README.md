# fill-me-up
## fillMeUp - food delivery

Dummy project for mocking food delivery services like Wolt or Glovo. 

Dependencies used:
* Node + express (http server)
* Mongoose (database)
* Pino (logger)
* zod (object parser and validator)

# How to run
`npm run dev` - development
